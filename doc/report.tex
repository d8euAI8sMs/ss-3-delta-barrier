\section{Постановка задачи}

Исследовать зависимость коэффициента прохождения частицы через барьер
%%
\begin{equation}\label{eq:barrier}
U(x) = V_0 \sum\limits_{i=0}^{N-1} {\delta\left(x - i a\right)} \textrm{,}
\end{equation}
%%
представляющий собой гребенку дельта-функций Дирака, от начальной энергии частицы. $L = (N - 1) a$ -- заданная общая ширина барьера. $V0$ -- мощность дельта-функций Дирака.

\section{Физическая модель}

Будем рассматривать инфинитное движение квантовой частицы известной массы и энергии в одномерном стационарном потенциале, заданном в виде последовательности дельта-барьеров одинаковой \enquote{высоты}, смещенных друг относительно друга на постоянную величину. При такой постановке задачи спектр возможных энергий частицы непрерывен.

\section{Математическая модель}

Стационарное уравнение Шредингера имеет вид:
%%
\begin{equation}\label{eq:schrodinger}
\frac{d^2\varphi(x)}{dx^2} + \frac{2m}{\hbar^2} \left(E - U(x)\right) \varphi(x) = 0 \textrm{,}
\end{equation}
%%
где $E$ -- энергия частицы, а $U(x)$ согласно задаче задается в виде \autoref{eq:barrier}.

В области вне барьера ($U(x) = 0 \Leftrightarrow x \leq x_l = 0 \cup x \geq x_r = L$) решением уравнения будет являться суперпозиция двух бегущих на встречу друг другу волн $\varphi(x) = A \exp\left(i k x\right) + B \exp\left(- i k x\right)$, где $k = \sqrt{\frac{2m}{\hbar^2}E}$.

Пусть на барьер налетает частица с волновой функцией единичной амплитуды. Тогда в области до барьера решение \autoref{eq:schrodinger} будет иметь вид:
%%
\begin{equation}\label{eq:before_barrier}
\varphi(x) = \exp\left(i k x\right) + R \exp\left(- i k x\right) \textrm{,}
\end{equation}
%%
$R$ -- коэффициент отражения от барьера. После барьера нет отраженной волны, поэтому решение \autoref{eq:schrodinger} получится в виде:
%%
\begin{equation}\label{eq:after_barrier}
\varphi(x) = T \exp\left(i k x\right) \textrm{,}
\end{equation}
%%
$T$ -- коэффициент пропускания.

Задача поиска коэффициента пропускания равносильна поиску $\left|\varphi(x)\right|_{x = x_r}$ при граничных условиях $A = 1$ на левой границе $x = x_l$ (на барьер налетает волна единичной амплитуды) и $B = 0$ на правой границе $x = x_r$ (отсутствие отраженной волны после барьера).

\section{Численная модель}

\subsection{Численный метод}

Для решения задачи воспользуемся методом прогонки. В простейшей постановке метод утверждает, что из граничных условий для уравнения вида
%%
\begin{equation}\label{eq:schrodinger_generic}
\varphi''(x) = p(x) \varphi(x) + q(x) \textrm{,}
\end{equation}
%%
заданных в виде
%%
\begin{equation}\label{eq:boundary_condition_generic}\begin{aligned}
\varphi'(x_l) &= \mu_0 \varphi(x_l) + \nu_0 \textrm{,} \\
\varphi'(x_r) &= \mu_1 \varphi(x_r) + \nu_1 \textrm{,}
\end{aligned}\end{equation}
%%
где значение $\varphi(x_r)$ не известно, можно получить начальные условия $\varphi(x_r)$, $\varphi'(x_r)$ через решение системы дифференциальных уравнений первого порядка
%%
\begin{equation}\begin{aligned}\label{eq:alpha_beta}
\alpha'(x) + \alpha^2(x) &= p(x) \textrm{,} \\
\beta'(x) + \alpha(x)\beta(x) &= q(x) \textrm{,}
\end{aligned}\end{equation}
%%
с начальными условиями $\alpha(0) = \mu_0$, $\beta(0) = \nu_0$:
%%
\begin{equation}\begin{aligned}
\varphi(x)\Big|_{x = x_r} &= \frac{\beta(x_r) - \nu_1}{\mu_1 - \alpha(x_r)} \textrm{,} \\
\varphi'(x)\Big|_{x = x_r} &= \frac{\mu_1 \beta(x_r) - \nu_1 \alpha(x_r)}{\mu_1 - \alpha(x_r)} \textrm{,}
\end{aligned}\end{equation}
%%

Задача решения ДУ второго порядка при заданных начальных условиях может решаться любыми подходящими методами решения ДУ, например методом Рунге-Кутты четвертого порядка.

Приведем граничные условия решаемой задачи к виду \autoref{eq:boundary_condition_generic}. Для чего продифференцируем \autoref{eq:before_barrier} и \autoref{eq:after_barrier} и исключим из них $R$ и $T$. Получим
%%
\begin{equation}\begin{aligned}
\varphi(x)\Big|_{x = x_l} &= - i k \varphi(x_l) + 2 i k \exp(i k x_l) \textrm{,} \\
\varphi'(x)\Big|_{x = x_r} &= i k \varphi(x_r) \textrm{.}
\end{aligned}\end{equation}
%%

\subsection{Выбор приближений}

Дельта-потенциал будем моделировать гауссовым куполом, достаточно узким, чтобы его можно было с определенной степенью точности считать дельта-функцией:
%%
\begin{equation}\begin{aligned}\label{eq:delta_approx}
\delta(x, x_0) \approx \frac{1}{\sqrt{2\pi} \sigma} \exp{\left(-\frac{(x - x_0)^2}{\sigma^2}\right)} \textrm{.}
\end{aligned}\end{equation}
%%

\subsection{Приведение к безразмерному виду}

В целях борьбы с погрешностями, вносимыми спецификой компьютерных вычислений, проведем обезразмеривание параметров задачи.

Введем новые относительные величины:
%%
\begin{equation}\begin{gathered}\label{eq:relative_vars}
\xi = \frac{x}{\xi_0} \textrm{, }
l = \frac{L}{\xi_0} \textrm{, }
e = \frac{2m}{\hbar^2} \frac{E}{e_0} \textrm{, }
v = \frac{2m}{\hbar^2 \xi_0} \frac{V_0}{v_0} \textrm{, } \\
\psi(\xi) = \varphi(x = \xi\xi_0) \textrm{,} \\
b(\xi) = \frac{2m}{\hbar^2} U(x = \xi\xi_0) = v \sum\limits_{i=0}^{N-1} {\delta\left(\xi - i \frac{l}{N-1}\right)} \textrm{.}
\end{gathered}\end{equation}
%%

Применим \autoref{eq:delta_approx} к \autoref{eq:relative_vars}, введем для удобства моделирования $\varkappa = \sigma / \frac{l}{N - 1}$, получим:
%%
\begin{equation}
\tilde{b}(\xi) = v \frac{1}{\sqrt{2\pi} \varkappa \frac{l}{N - 1}} \sum\limits_{i=0}^{N-1} {\exp{\left(-\frac{\left(\xi - i \frac{l}{N-1}\right)^2}{\left(\varkappa \frac{l}{N-1}\right)^2}\right)}} \textrm{.}
\end{equation}
%%

Тогда, окончательно, \autoref{eq:schrodinger} будет иметь вид:
%%
\begin{equation}\begin{aligned}
\psi''(\xi) = - \xi_0^2 \left(e - \tilde{b}(\xi)\right) \psi(\xi) \textrm{.}
\end{aligned}\end{equation}
%%

При моделировании можно принять $v_0 = e_0 = 1$, $\xi_0 = L$. Левую границу интервала моделирования $\xi_l$ целесообразно брать равной $\xi_l = - 3..6 \varkappa$, где влияние первого дельта-барьера гребенки дельта-барьеров в приближении гауссового купола практически отсутствует. Симметрично, правую границу брать равной $\xi_r = l + 3..6 \varkappa = 1 + 3..6 \varkappa$. Выбор различных единиц для измерения энергии частицы и высоты барьера обусловлен соображениями удобства -- такой выбор обеспечивает независимость высоты барьера от его ширины, что позволяет проследить зависимость коэффициента прохождения от ширины барьера.
